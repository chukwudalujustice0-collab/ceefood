# CeeFood

A modern Flutter food delivery application with restaurant browsing, cart management, and real-time order tracking.

## Features
- Home screen with featured restaurants.
- Detailed restaurant menu browsing.
- Interactive cart system using Provider.
- Order tracking status timeline.
- Material 3 design with Google Fonts.

## Getting Started
To run this project:
1. Ensure Flutter is installed.
2. Run `flutter pub get`.
3. Run `flutter run`.
